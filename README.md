# Moss WPF

MossWPF is a Windows desktop client for creating and managing submissions to Stanford’s plagiarism-detection service known as MOSS (**M**easure **O**f **S**oftware **S**imilarity). The MOSS algorithm is excellent at what it does, however its usage can be a bit daunting. MossWPF provides instructors and professors an easy to use interface and additional features in order to get the most out of the MOSS service.
